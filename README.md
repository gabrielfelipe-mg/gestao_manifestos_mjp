# gestao_manifestos_mjp
# 📦 Sistema de Gestão de Manifestos - MJP Transportes

Sistema desenvolvido em **Python** e **MySQL** para automatizar o controle de manifestos e CTEs no setor de logística. O objetivo foi eliminar processos manuais e centralizar as informações.

## 🛠️ Tecnologias
- **Linguagem:** Python 3
- **Banco de Dados:** MySQL
- **Bibliotecas:** `mysql-connector-python`, `datetime`, `time`

## 🚀 Funcionalidades
- **CRUD Completo:** Cadastro, listagem, atualização e exclusão de registros.
- **Automação:** Data de registro gerada automaticamente pelo sistema.
- **Segurança:** Tratamento de erros com blocos `try/except`.

## ⚙️ Como executar
1. Clone o repositório.
2. Crie o banco de dados `cadastro` usando o script SQL incluso.
3. Instale a dependência: `pip install mysql-connector-python`
4. Altere a senha no arquivo principal e execute: `python sistema_mjp.py`
